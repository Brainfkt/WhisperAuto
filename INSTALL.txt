WisperAuto - guide d'installation rapide
=======================================

Objectif
--------
WisperAuto transcrit localement des fichiers audio puis produit :
- une transcription brute ;
- une transcription intelligente par LLM local ;
- un compte rendu structure.

Les audios ne sont pas envoyes en ligne. Internet est utilise uniquement si
l'utilisateur lance explicitement une installation ou un telechargement depuis
l'interface.


Installation recommandee Windows
--------------------------------
1. Installer Python 3.11 ou 3.12 :
   https://www.python.org/downloads/windows/

2. Placer le projet dans un dossier simple, par exemple :
   C:\WisperAuto

3. Double-cliquer :
   start_whisper.bat

Le lanceur cree un venv local ".venv" si besoin. Il privilegie Python 3.11 ou
3.12 et evite Python global Anaconda / Python 3.13 pour le deploiement client.


Premiere configuration depuis l'interface
-----------------------------------------
Ouvrir Parametres > Telechargements, puis installer dans cet ordre :

1. FFmpeg / ffprobe
2. Moteur faster-whisper, ou garder le moteur Auto
3. Modele moteur
4. Post-traitement LLM local
5. Modele post-traitement LLM GGUF

Modele conseille :
- large-v3-turbo pour une bonne precision avec une UX locale raisonnable ;
- medium si le poste est lent ou sans GPU.


Installation macOS / Linux
--------------------------
Depuis le dossier du projet :

python3.12 -m venv .venv
source .venv/bin/activate
python auto_transcribe.py

Puis installer FFmpeg, moteur, modeles et LLM depuis Parametres >
Telechargements.

Notes :
- Python 3.11 fonctionne aussi.
- Sur Mac Apple Silicon, tester le moteur MLX Mac si disponible.
- Sur Linux, installer le paquet systeme Python Tk si l'interface ne s'ouvre pas.


Utilisation
-----------
1. Cliquer Ajouter.
2. Choisir un ou plusieurs fichiers audio.
3. Cliquer Transcrire ou Tout.
4. Suivre la progression en bas de l'application.
5. Choisir le mode de sortie : brute, intelligente ou compte rendu.
6. Utiliser Copier, Exporter ou Dossier.

Pendant le post-traitement intelligent, l'application affiche le chargement du
LLM local, le segment en cours, le temps ecoule et une ETA quand elle est fiable.

Le bouton Annuler demande l'arret du traitement. Pendant un appel LLM long,
l'annulation est appliquee a la fin du segment en cours.

Le bouton Supprimer retire l'entree d'historique et les fichiers texte generes.
Les audios archives ou echoues sont conserves pour eviter une perte.


Telechargements Hugging Face plus rapides
-----------------------------------------
Si un modele se telecharge lentement :

1. Ouvrir Parametres > Telechargements.
2. Cliquer Creer token.
3. Creer un token Hugging Face en lecture seule.
4. Coller le token dans HF token puis Enregistrer.
5. Activer Telechargement rapide Xet si souhaite.

Le token est stocke localement dans settings.json et n'est pas affiche dans les
commandes du journal.


Dossier de donnees
------------------
Par defaut :
%USERPROFILE%\Documents\WisperAuto

Sous macOS / Linux :
~/Documents/WisperAuto

Principaux sous-dossiers :
- inbox : fichiers ajoutes avant traitement ;
- processed : audios traites ;
- failed : audios en erreur ;
- outbox : transcriptions ;
- logs : historique local ;
- models : modeles locaux.

Pour changer de dossier sous Windows :
set WISPERAUTO_HOME=D:\WisperAutoData
start_whisper.bat


Depannage
---------
FFmpeg introuvable :
- installer FFmpeg depuis Parametres > Telechargements ;
- relancer WisperAuto si le PATH a change.

Modele moteur absent :
- choisir le moteur et le modele dans Parametres ;
- cliquer Telecharger modele moteur.

LLM intelligent indisponible :
- installer Post-traitement LLM local ;
- telecharger Modele post-traitement LLM GGUF ;
- la transcription brute reste disponible en cas d'erreur.

Transcription lente :
- utiliser le profil Rapide ;
- essayer le modele medium ;
- lancer Tester performances ;
- sur Mac Apple Silicon, tester MLX Mac ;
- en CPU-only, tester whisper.cpp avec un modele quantized.

DSS / DS2 illisible :
- certains fichiers DSS Pro ou DS2 ne sont pas pris en charge par FFmpeg ;
- convertir avec le logiciel du dictaphone avant import si necessaire.


Mode console
------------
Interface graphique :
python auto_transcribe.py

Transcrire un fichier :
python auto_transcribe.py --once "C:\chemin\fichier.mp3"

Surveiller le dossier inbox :
python auto_transcribe.py --watch
