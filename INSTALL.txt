Formats supportés :

.dss
.mp3
.wav
.m4a

Points de vigilance :
- Modèle large-v3 performant mais lent -> passage en 'medium' voire 'small' si trop long > compromis qualité
- FFmpeg supporte bien le DSS, mais mal le DS2/DSS Pro -> doc : https://trac.ffmpeg.org/ticket/6091
- le script attend que la taille du fichier se stabilise, donc c’est déjà une bonne protection, mais Watchdog peut déclencher dès le début de la copie, pas forcément quand elle est finie donc toujours vérifier le résultat.
    Sinon on oublie le reverse engineering et on installe le logiciel https://audiosupport.omsystem.com/en/product-category/software/ (solution plus manuelle)
